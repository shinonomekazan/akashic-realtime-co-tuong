# akashic-realtime-co-tuong

## 利用方法
```sh
npm install
```

### 動作確認方法

以下のアクションを実行した後、ブラウザで `http://localhost:3000/game/` にアクセスすることでゲームを実行できます。

* `akashic install`

* `akashic serve`
